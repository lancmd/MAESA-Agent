# ENVI 分类精度评价流程

精度评价用于判断分类图能否支持面积统计、土地利用变化、PLUS 和生态服务分析。训练精度不能替代独立验证精度。

## 1. 验证数据

验证数据可来自独立 ROI、高分辨率影像、无人机、野外点或可靠历史资料。要求：

- 与训练样本空间独立且不重叠；
- 覆盖所有类别及研究区主要环境梯度；
- 记录样本来源、判读日期和不确定性；
- 多算法比较时固定同一验证集；
- 连续大 ROI 内的相邻像元不应被误认为完全独立样本。

建议每类至少准备 30 个空间分散的验证单位；稀有类别可采用分层抽样增加样本。样本数应依据目标置信度、类别比例和允许误差确定，不能仅靠统一像元阈值。

## 2. ENVI 操作

使用 ROI 评价：

`Toolbox → Classification → Post Classification → Confusion Matrix Using Ground Truth ROIs`

1. 选择最终 Classification Raster；
2. 选择独立 Ground Truth ROIs；
3. 将验证 ROI 名称与分类结果类别逐一匹配；
4. 导出完整混淆矩阵和精度报告。

使用 Ground Truth Image 时，真值图与分类图必须具有完全一致的行列数、像元大小、范围和空间参考。

## 3. 必报指标

| 指标 | 含义 | 重点解释 |
|---|---|---|
| Overall Accuracy | 正确样本占全部验证样本的比例 | 易受优势类别主导 |
| Producer's Accuracy | 真值类别被正确识别的比例 | 对应漏分/遗漏误差 |
| User's Accuracy | 地图中某类实际正确的比例 | 对应错分/委托误差 |
| Confusion Matrix | 真值类别与地图类别的交叉计数 | 用于定位具体混淆方向 |
| Kappa | 超越随机一致性的传统指标 | 可报告，但不应单独决定质量 |

条件允许时，同时计算每类 F1 分数，并报告验证样本数量和抽样设计。

## 4. 结果判定

总体精度 80% 或 85%、Kappa 0.75 或 0.80 可作为项目目标示例，但不是所有矿区的通用合格线。应在研究开始前根据用途设定阈值，并重点检查：

- 沉陷积水、建设用地、裸地/工矿用地等关键类别是否达到用途要求；
- 是否存在总体精度较高但小类别精度很低的情况；
- 未分类像元是否集中在某类或某个空间区域；
- 后处理是否只改善外观，却降低小斑块类别精度。

用于面积估算时，若采用分层概率抽样，应在 ENVI 报告之外计算面积校正结果及置信区间。

## 5. 常见混淆与修正

| 混淆 | 优先检查 |
|---|---|
| 建设用地 ↔ 裸地/工矿用地 | ROI 纯度、纹理、SWIR/NDBI、施工地定义 |
| 收割后耕地 ↔ 裸地 | 影像时相、耕地边界和历史影像 |
| 草地 ↔ 低覆盖耕地 | 物候、纹理、样本空间覆盖 |
| 林地 ↔ 高覆盖草地 | 阴影、冠层纹理和季节差异 |
| 沉陷积水 ↔ 自然水体 | 采矿范围、历史水体和沉陷资料 |
| 阴影 ↔ 水体 | 多波段响应、地形和邻域关系 |

修正后必须重新运行完整精度评价，并记录训练样本版本、分类器参数和后处理参数。不得只挑选精度最高的一次结果而省略其他试验条件。
